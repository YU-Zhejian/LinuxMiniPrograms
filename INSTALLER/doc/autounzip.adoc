= autounzip(1)

== NAME

autounzip - An automatic file decompression tool.

== SYNOPSIS

autounzip [source] [options]

== DESCRIPTION

Extract [source]. If the archive is SPLITTED by `autozip`, suffix like "001" or "z01" should not be added. HOWEVER, for rar you should indicate the FIRST part. For example,  "r1.part1.rar". If [source] is omitted, the prgram will check the algorithms installed on your computer and give back a list of available extension name.

== OPTIONS

include::include/cmnopt.adoc[]

`--force-parallel:[location]` Force to use GUN parallel at [location]. This will not affect pgzip, XZ Utils or other algorithms which do not require GNU Parallel.

`--force` Remove the target if exists.
`--remove` Remove [sorce] if success.
