= azman(1)

== NAME

azman - AutoZip file Manager.

== SYNOPSIS

azman [subcommand] [source] [options]

== DESCRIPTION

Use [subcommand] on [source]. The subcommands will be introduced bellow. If one of the [subcommand] or [source] is omitted, will check the algorithms installed on your computer and give back a list of available extension name.
