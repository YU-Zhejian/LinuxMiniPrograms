= lcman(1)

== NAME

lcman - LiveChat History Manager

== SYNOPSIS

lcman [OPTIONS]

== DESCRIPTION

Filter the history by OPTIONS.

See OPTIONS section.

== OPTIONS

include::include/cmnopt.adoc[]

* `-f:[STRING]|--from:[STRING]` Search in history with sender [STRING].

* `-m:[STRING]|--message:[STRING]` Search in history with message [STRING].

* `-s|--exclude-system` Exclude message from SYSTEM.

* `-e:[STRING]|--exclude:[STRING]` Search in history without sender [STRING].

== COMPATIBILITY

=== BSD

Tested with all functions except coloring supported. Use `--more:cat` instead of the default `more` on BSD.

=== CYGWin

Tested with all functions supported.
