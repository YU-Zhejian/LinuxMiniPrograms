= bhs(1)

== NAME

bhs - Bash History Searcher.

== SYNOPSIS

`bhs [STRING] [OPTIONS]`

== DESCRIPTION

Search `[STRING]` in ~/.bash_history with 'grep'.

== OPTIONS

include::include/cmnopt.adoc[]

These examples are tested under CYGWin.

```
admin@DESKTOP-FFEV0AF ~
```
