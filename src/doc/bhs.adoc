= bhs(1)

== NAME

bhs - Bash History Searcher.

== SYNOPSIS

`bhs [[STRING] [STRING] ...] [OPTIONS]`

== DESCRIPTION

Search `[STRING]` in ~/.bash_history with 'grep'. Search for multiple targets are done spontaneously.

This program can be replaced by `history | grep XXXXXX`.
