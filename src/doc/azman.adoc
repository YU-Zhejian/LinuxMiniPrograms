= azman(1)

== NAME

azlist - AutoZip file viewer.

== SYNOPSIS

`azlist [SOURCE1] [[SOURCE2]...]`

== DESCRIPTION

List all files and folders inside `[SOURCE]` in a verbose format.

WARNING: This program do not support archives with extension `.lzma`, `.lz` and `.bz2`.

== OPTIONS

* `-p[:[thread]]|--parallel[:[thread]]` Use `[thread]` threads when extracting the archive. `[thread]` should not be higher than the number of cores installed.
If `[thread]` is omitted, will use the number of cores installed.
