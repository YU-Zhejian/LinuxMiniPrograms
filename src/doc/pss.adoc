= pss(1)

== NAME

pss - A program to search Process Snapshots

== SYNOPSIS

pss [STRING] [[STRING2]...] [OPTIONS]

== DESCRIPTION

Search [STRING] in Process Snapshots.
If [STRING] is omitted, the program will list all Process Snapshots
