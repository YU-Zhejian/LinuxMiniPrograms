== NAME

libdo_monitor - Monitor a process

== SYNOPSIS

libdo_monitor [LIBDO_PID] [PPID]

== DESCRIPTION

Monitor `[LIBDO_PID]` and its sub-processes/threads by watching its CPU, memory and disk IO. We also use `pstree` to expand `[PPID]`.

== OPTIONS

None.
