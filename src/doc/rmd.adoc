= rmd(1)

== NAME

rmd - YuZJLab RMarkDown Compiler.

== SYNOPSIS

`rmd [filename] [[filename] ...] [OPTIONS]`

== DESCRIPTION

A wrapper which compiles RMarkdown file `[filename]` with `knitr`.
