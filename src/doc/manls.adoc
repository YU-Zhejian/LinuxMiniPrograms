= manls(1)

== NAME

manls - A program to search `${MANPATH}` variables

== SYNOPSIS

manls [STRING] [[STRING2]...] [OPTIONS]

== DESCRIPTION

Search `[STRING]` in `${MANPATH}`.
If `[STRING]` is omitted, the program will list all contents in each valid `${MANPATH}`.

== OPTIONS

include::include/cmnopt.adoc[]

* `-l|--list` List all valid manpath variable.
