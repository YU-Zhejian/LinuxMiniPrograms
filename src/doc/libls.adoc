= libls(1)

== NAME

libls - A program to search C lib paths

== SYNOPSIS

libls [STRING] [[STRING2]...] [OPTIONS]

== DESCRIPTION

Search `[STRING]` in `${LIBRARY_PATH}` `${LD_LIBRARY_PATH}` and other system lib paths.
If `[STRING]` is omitted, the program will list all contents in each valid lib paths.

== OPTIONS

include::include/cmnopt.adoc[]

* `-l|--list` List all valid C include paths.

* `-s|--dynamic-only` Dynamically linked libs (`.so`) only.

* `-a|--static-only` Statically linked libs (`.a`) only.
