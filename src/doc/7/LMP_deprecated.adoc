= LMP_deprecated(1)

== NAME

LMP_deprecated - Deprecated functions of LinuxMiniPrograms

== SYNOPSIS

`man LMP_deprecated`

== DESCRIPTION

Here gives a list of deprecated programs in YuZJLab LinuxMiniPrograms.

. lomux
.. `bin/iomux`
.. `bin/iomux_a`
.. `bin/iomux_r`
. ylmktbl (part)
.. `bin/exec/ylmktbl.sh`
. libdosh
.. `bin/libdosh`
. libdoman_upgrade
.. `bin/libdoman_upgrade`
. b16c
.. `bin/b16c`
.. `bin/exec/b16c.co`
.. `bin/exec/b16c.py`
.. `src/C/bin/b16c.c`
. git-mirror
.. `bin/git-mirror`
.. `bin/exec/gitm/*`

These files should appear in `src/DEPRECATED`, and will no longer be updated.
