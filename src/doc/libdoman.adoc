= libdoman(1)

== NAME

libdoman - LibDO log manager.

== SYNOPSIS

`libdoman [log] [[log2] ...] [OPTIONS]`

== DESCRIPTION

Decode LibDO output information from logs. Please note that this program do not take additional logs generated by LibDO3.

== OPTIONS

include::include/cmnopt.adoc[]
* `-o:[item]|--output:[item]` Get more information about `[item]`. The numbering of items started from 1.
* `-m|--machine` Use machine mode, output tables in Tab Separated Values (TSV). Capable for pipelines.
