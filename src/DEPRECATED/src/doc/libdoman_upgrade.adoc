= libdoman_upgrade(1)

== NAME

libdoman_upgrade - LibDO log updater.

== SYNOPSIS

`libdoman_upgrade [log1] [[log2] ...]`

== DESCRIPTION

Update libdo logs from 1 to 2.

WARNING: This program is no longer monitored and updates may not be provided!

== COMPATIBILITY

=== BSD

Tested with all functions supported.

=== Cygwin

Tested with all functions supported.
